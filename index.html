Git Commands

  • git init: Initialize a new Git repository.
  • git clone [url]: Clone an existing repository from a URL.
  • git add [file]: Stage changes for the next commit.
  • git add .: Stage all the changes in the working directory.
  • git commit -m "[message]": Commit staged changes with a descriptive message.
  • git status: Check the status of your working directory and staging area.
  • git push: Push committed changes to a remote repository.
  • git pull: Fetch and merge changes from a remote repository.
  • git branch: List, create, or delete branches.
  • git checkout [branch]: Switch to a different branch.
  • git merge [branch]: Merge changes from one branch into the current branch.,and if the changes are made in same place in same file , then we see an option to select what change do we want,like changes frm branch1 or branch2 or we want to keep both
